# The Movie Project

## How to run in local

#### 1. Clone the repo

```
git clone https://github.com/Chaithanya065/themovieproject.git

cd themovieproject
```

#### 2. Install dependencies

```
npm install
```

#### 3. Set environment variables

```
cp .env.example .env
```

#### 4. Populate the .env file with your own configuration

#### 5. Run it locally

```
npm run dev
```
